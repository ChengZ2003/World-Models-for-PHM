# Reproduction Index

The reproduction index records evidence and obstacles; it is not a claim that all methods have been reimplemented. Reports belong in `reports/` and should use [`REPRODUCTION_TEMPLATE.md`](REPRODUCTION_TEMPLATE.md).

Allowed statuses are: Not evaluated, Code available, Environment built, Partial reproduction, Reproduced, Failed reproduction, Blocked by unavailable data, and Blocked by missing implementation details.

Never describe an unsuccessful run as reproduced, present an approximate implementation as official, hide data-split differences, or omit failed experiments.
