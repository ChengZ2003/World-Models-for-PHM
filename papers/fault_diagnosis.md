# Fault Diagnosis

This page covers work that identifies fault type, location, or cause and has a meaningful temporal-dynamics connection. Static classifiers are outside the core scope.

## Inclusion Considerations

Diagnosis work belongs in the core scope only when temporal state or transition modeling materially supports fault attribution.
