# Anomaly Detection

This page covers models that use learned current or predicted system behavior to identify abnormality in an observed interval. It distinguishes detection from future-horizon early warning.

## Inclusion Considerations

The method should connect a learned state or dynamics mechanism to current-window anomaly evidence. Pure static classification belongs outside the core scope.
