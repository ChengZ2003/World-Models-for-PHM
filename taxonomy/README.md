# Taxonomy

The taxonomy describes methods along separable dimensions rather than treating “world model” as a binary keyword.

1. Apply the [`inclusion_criteria.md`](inclusion_criteria.md).
2. Assign a scope in [`world_model_taxonomy.md`](world_model_taxonomy.md).
3. Record downstream tasks, representation, transition mechanism, learning objectives, uncertainty treatment, and domains in the corresponding files.

Classification is provisional when evidence is incomplete. Disagreements should be recorded rather than silently normalized.
