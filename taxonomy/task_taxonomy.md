# Task Taxonomy

| Task | Target time | Output |
| --- | --- | --- |
| Anomaly detection | Current/observed interval | Abnormality score or label |
| Anomaly prediction / early warning | Future horizon | Risk, alarm, or event probability |
| Fault diagnosis | Current or inferred cause | Type, location, or cause |
| Health assessment | Current/latent state | Health state or index |
| Fault prognosis | Future evolution | Degradation or failure trajectory |
| RUL prediction | Time to failure condition | Point or predictive distribution |
| Maintenance decision | Future intervention | Timing or action policy |

Papers can address multiple tasks, but the evaluation target and time orientation should remain explicit.
