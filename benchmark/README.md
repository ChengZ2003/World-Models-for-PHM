# Benchmark Protocol Exploration

This directory is a future lightweight protocol and evaluation layer for world-model-oriented PHM questions. It is not a completed benchmark or a replacement for established PHM platforms, anomaly-detection suites, or dataset tooling.

Current work is limited to protocol drafts and an engineering roadmap. Future implementation should reuse existing loaders and frameworks where practical.

> Dataset redistribution is not assumed. Users must follow the original dataset licenses and access procedures.
