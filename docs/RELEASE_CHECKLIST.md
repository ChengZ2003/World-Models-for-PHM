# Release Checklist

- [ ] Version is consistent in `CITATION.cff` and `CHANGELOG.md`.
- [ ] Metadata validation passes.
- [ ] Generated paper and repository tables are current.
- [ ] Markdown relative links resolve.
- [ ] New factual records are manually verified.
- [ ] Scope and status language reflects completed work only.
- [ ] Reproduction claims have inspectable evidence.
- [ ] Dataset licenses and access restrictions are respected.
- [ ] Changelog describes user-visible changes.
- [ ] Public publication checklist is complete.
