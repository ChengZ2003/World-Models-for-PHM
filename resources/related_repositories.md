# Related Repositories

This repository builds on and complements existing efforts in general world models, predictive maintenance, PHM benchmarking, multivariate anomaly detection, and RUL literature curation. It does not replace their training infrastructure, benchmark implementations, or broader literature coverage.

Only manually verified entries should be published as factual records. The structured source is [`../data/repositories.csv`](../data/repositories.csv), and the verified table is generated at [`generated_related_repositories.md`](generated_related_repositories.md).

## Required Metadata

Each record should include repository name, focus, main strength, relationship to this repository, URL, last manually checked date, and verification status.
