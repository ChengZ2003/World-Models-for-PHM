# Related Repositories

This repository builds on and complements existing efforts in general world models, predictive maintenance, PHM benchmarking, multivariate anomaly detection, and RUL literature curation. It does not replace their training infrastructure, benchmark implementations, or broader literature coverage.

The structured source of truth is [`../data/repositories.csv`](../data/repositories.csv). The verified-only landscape table is generated at [`generated_related_repositories.md`](generated_related_repositories.md). Do not maintain repository data rows on this handwritten page.

## Selection Principles

- Include a resource only when its current focus and relationship to this project have been manually checked.
- Prefer official repository URLs and describe scope without promotional claims.
- Record the last checked date and keep uncertain entries unverified.
- Treat inclusion as ecosystem mapping, not endorsement.

## Category Definitions

- **General World Model Collections:** broad collections centered on learned environment or system dynamics.
- **PHM Frameworks and Benchmarks:** infrastructure for PHM training, evaluation, or dataset access.
- **Time-Series Anomaly Detection Benchmarks:** evaluation suites focused on anomaly detection protocols.
- **RUL and Prognostics Resources:** literature, implementations, or infrastructure centered on degradation and remaining life.
- **Predictive Maintenance Collections:** resources covering maintenance prediction and decision support.
- **Time-Series Foundation Model Resources:** general pretrained time-series resources relevant to, but not automatically classified as, PHM world models.

## Required Metadata

Each record must document repository name, category, current focus, main scope, relationship to this repository, official URL, last manually checked date, and verification status.
